Meu lindo crochet - Roxanne Coelho

Olá a tod@s! O meu nome é Roxanne Coelho e criei este site para ensinar crochet de forma didática e divertida!

Sou mestre em Biologia - Ramo Ecologia Aplicada pela Universidade de Aveiro e crochet é um dos meus hobbies preferidos. Espero que gostem e acima de tudo que criem os vossos projetos!

Para começar no mundo de crochet é preciso:

  • Linhas de lã (ou acrílico);
  • Agulha de crochet adequada à espessura da linha;
  • Tesoura
  • Agulha de malha para finalizar os projetos
  • Muita paciência e força de vontade!

Vamos começar com um projeto simples: fazer um quadrado! Começamos com:

  1. Ponto corrente
  2. Ponto simples
  3. Virar o projeto
  4. Ponto simples
  5. Finalizar o projeto com ponto baixíssimo
